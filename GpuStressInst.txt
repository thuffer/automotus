GPU stress Instructions 

1-Open the LXTerminal and Navigate to the Desktop folder by entering on the command line: 
	cd Desktop/

2-On the command line, download the program by entering:
	 wget https://github.com/thuffer/automotus/raw/main/matrixMul

3-Get permission to run program by entering:
	 chmod u+x matrixMul

4-Run the program by entering: 
	./matrixMul -wA=6400 -hA=1000 -wB=1000 -hB=6400
